One key chord

Karabiner-Elements maps Option+V to the helper, so dedented paste feels like a native paste command.

Single executable

GitHub Releases publish one universal macOS binary named dedent-paste for Apple Silicon and Intel Macs.

Plain text, UTF-8 safe

The clipboard is read as text, written back as UTF-8, and pasted through macOS System Events.

Paste clean, keep the source intact For snippets, Markdown, prompts, and notes that need to move without extra indentation.

dedent-paste only changes the clipboard text you paste with Option+V. Keep the original source when exact indentation matters, and allow Karabiner-Elements in macOS Accessibility so the paste automation can run.
